Files to download can be specified via the configuration input.

Force to follow redirects If false only redirect in scope of the same protocol are followed. If true redirect to any any address is followed. Can be used to for example follow redirect from http to https. Use UTF-8 encoding for redirect Can be used together with 'Force to follow redirects'. Interpret Location header using UTF-8. Skip on error If set and download failed, the file is skipped. If not set and download failed the component fail. Detailed log If set, additional information is logged for every download. Encode input URL If true (percent) encode input URL, can be used when URL contains non-ASCI characters.
