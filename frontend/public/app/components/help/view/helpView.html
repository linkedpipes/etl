Feedback

Bugs, errors and feature requests can be reported as issues on the GitHub page. To report a bug please include the used version of LinkedPipes ETL (see section Version below).

In case of a problem with the web client, check the browser developer console for errors, and provide their list as part of the issue description.

If your are experiencing problems with a specific pipeline, feel free to upload it to the demo instance and provide the link to the pipeline (execution) in the issue description.
